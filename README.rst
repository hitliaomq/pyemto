pyemto
======

Easily generate input files and corresponding batch scripts for the EMTO
(Exact Muffin-Tin Orbitals) solid-state physics software.

Installation
------------

::

   git clone https://github.com/hpleva/pyemto.git
   cd pyemto
   python setup.py install
